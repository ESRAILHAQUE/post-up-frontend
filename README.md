**A modern, responsive frontend for PostUp - Premium Guest Posting Platform**

This is the frontend application for PostUp, built with Next.js, TypeScript, and Tailwind CSS. It provides an intuitive interface for businesses to discover and purchase premium guest posting opportunities on high-authority websites.

## Overview

PostUp connects businesses with premium guest posting opportunities on high-authority websites. This frontend provides a seamless user experience for browsing sites, managing orders, and accessing admin features.
